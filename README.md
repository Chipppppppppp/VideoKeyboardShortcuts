# YT-Like Shortcuts

## 概要

YouTube ライクなキーボードショートカットを様々なサイトで。アイコンをクリックして動作させたいサイトを追加します。ニコニコ動画 (nicovideo.jp) などがサポートされています。
追加ショートカット
- Shift + \ ("_"):  再生スピードをリセット
- Shift + 右/左矢印キー: 30 秒進む/戻る
- Shift + L/J: 60 秒進む/戻る
YouTube に元から存在するショートカットは以下のスクリーンショットから確認できます。(不要なものはサポートされていません)
